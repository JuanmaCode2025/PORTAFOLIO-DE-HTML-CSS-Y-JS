Diseño de Factura

Proyecto centrado en el uso avanzado de elementos "table" en HTML5 para construir la estructura de una factura comercial. El objetivo fue demostrar la habilidad para manejar la maquetación de datos tabulares complejos utilizando "thead, tbody, tr, th, y td, " incluyendo el uso de atributos como colspan y rowspan para alinear y fusionar celdas. El diseño se estilizó con CSS3 para asegurar una presentación profesional y una separación clara de la estructura y el estilo.

Página Web de Viaje

Implementación de una página web estática con HTML y CSS que simula una plataforma de reserva/planificación de viajes. Pokedex

Desarrollo de una Pokédex estática/interactiva construida con HTML, CSS y JavaScript (ES6+). El núcleo del proyecto es la gestión de llamadas asíncronas para consumir datos de la PokeAPI pública, demostrando la habilidad para realizar peticiones fetch y manejar Promesas (Promises) o async/await. El sistema procesa y renderiza dinámicamente información detallada de Pokémon (estadísticas base, tipos, imágenes y habilidades) en una interfaz de usuario limpia y moderna.
